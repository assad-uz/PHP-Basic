Q: mysql database id int, name varchar — এখানে int আর varchar মানে কী?

A: এগুলো হলো Data Types (ডেটার ধরন)।

- ডেটাবেইসে প্রতিটি কলামের (যেমন: id, name) জন্য আপনাকে জানাতে হয় — “এই ঘরটিতে কেমন ধরনের ডেটা থাকবে?”
- এই ধরনের তথ্য বোঝাতে আমরা data types ব্যবহার করি।

Ex:

▶ id INT:

- id নামের ঘরটি সংখ্যা রাখবে।
- INT = Integer অর্থাৎ পুরো সংখ্যা (যেমন: 1, 5, 100, -20)
- এটি সাধারণত ইউজার আইডি, প্রোডাক্ট আইডি ইত্যাদির জন্য ব্যবহার হয়।

▶ name VARCHAR(50):

- name নামের ঘরটি লেখা বা টেক্সট রাখবে।
- VARCHAR = Variable-length Character (মানে: যতটুকু দরকার ততটুকু জায়গা নিবে)
- VARCHAR(50) = সর্বোচ্চ ৫০ অক্ষরের লেখা রাখা যাবে।


▶ অন্য Data Type এর উদাহরণ:

Data Type	                     মানে কী	                কবে ব্যবহার হয়

INT	                             পূর্ণ সংখ্যা	               বয়স, আইডি, পয়েন্ট
VARCHAR(n)	                 লেখা (n অক্ষর পর্যন্ত)	           নাম, ঠিকানা, ইমেইল
TEXT	                         বড় লেখা                    	মন্তব্য, বায়ো
DATE	                           তারিখ	              জন্ম তারিখ, প্রকাশ তারিখ
FLOAT / DOUBLE	                দশমিক সংখ্যা	              দাম, ওজন, গড়
